🚀 КАК ЗАПУСТИТЬ
═══════════════════════════════════════════════════════════════════

1️⃣  Получите токен у @BotFather в Telegram
2️⃣  Создайте файл .env с токеном: BOT_TOKEN=ваш_токен
3️⃣  Установите зависимости: pip install -r requirements.txt
4️⃣  Запустите бота: python main.py
5️⃣  Найдите бота в Telegram и напишите /start

Подробности в файле QUICKSTART.txt
