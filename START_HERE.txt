⚡ БЫСТРЫЙ ЗАПУСК (краткая версия):
───────────────────────────────────────────────────────────────────

1. Получите токен у @BotFather в Telegram
2. Создайте файл .env с вашим токеном:
   
   BOT_TOKEN=ваш_токен_здесь

3. Установите зависимости:
   
   pip install -r requirements.txt

4. Запустите бота:
   
   python main.py

5. Найдите бота в Telegram и напишите /start
